
### `!!`连续两次取反
将变量转换成它对应的布尔值，一般用来判断某个元素是否存在

1.!![]              true；                                            ![]             false
2.!!{}              true;                                              !{}            false
3.!!false           false;                                             !false         true
4.!!true            true;                                               !true         false
5.!!undefined       false;                                             !undefined     true
6.!!null            false;                                              !null         true
7.!!""              false;                                             !""            true
8.!!0               false                                               !0        true


### `~~`取整
