git常用命令
===========

### 创建、提交、提交到远程库

`git init` ： 本地git初始化

`git add README.md`

`git commit -m "first commit"`

`git remote add origin git@github.com:joeny3154/note.git`

`git push -u origin master`

### 提交到远程库（本地项目已经存在）

`git remote add origin git@github.com:joeny3154/note.git`

`git push -u origin master`
